# Pasta de Imagens

## Foto de Perfil

Para adicionar sua foto de perfil ao portfólio:

1. Coloque sua foto nesta pasta com o nome: `profile.jpg`
2. Formatos suportados: JPG, PNG, WEBP
3. Tamanho recomendado: mínimo 400x400 pixels (quadrada)
4. A imagem será exibida em formato circular (160x160 pixels)

### Fallback

Se a imagem não for encontrada, será usado um avatar gerado automaticamente como fallback.
